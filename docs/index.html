flexgrid

A simple grid system for all your layouting needs.

  • Getting started
  • Docs

1. Get the code

Using cdn

The easiest way to get started. Drop this into your page's head section and you're good to go. <link href="" rel="stylesheet" type="text/css">

Using bower

You can also get the latest release using bower. This release contains source files as well as the compiled CSS. bower install flexgrid --save

2. Create a row with some columns

Next just drop in the flexgrid classes in your HTML. <div class="row">
  <div class="col col--4">...</div>
  <div class="col col--4">...</div>
  <div class="col col--4">...</div>
</div>
